# ---新增---  
1.通过配置文件读取账号密码，无需再每次手工输入。 
2.输入过的fofa命令会有历史记录，方便快捷(暂不支持软件内删除，只能到txt删除)。 

# ---主要功能---  
1.支持批量fofa pro搜索； 
2.支持保存&导入txt或csv；   
3.支持查看搜索语法；   
4.支持各列的排序；   
5.支持双击或者多选回车批量打开网站；   
6.支持批量导入fofa语句，支持批量搜索； 
7.支配配置文件读取账号密码； 
8.支持在线或者批量导入icon图片，通过icon hash搜索资产。
